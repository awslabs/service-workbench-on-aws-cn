Enable Active Directory Authentication

To enable the Active Directory authentication, perform the following actions:

  1. Using Amazon Cognito on the AWS Management Console, create a Cognito User Pool. The name of the pool must be <stage>-<solution_name>-userPool, where stage and solution_name are as configured in the main configuration file.
  2. Gather the relying party information such as User Pool ID, Relying Party ID, User Pool Signing Cert, for the Active Directory integration.
  3. Run the following script from the root folder of the project: 
scripts/get-relying-party.sh
  1. Provide the output of this script to your Active Directory administrator. An example of the output is described in Figure 19.
