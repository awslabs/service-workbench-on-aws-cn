本文档的这一部分旨在概述如何操作作为此解决方案的一部分提供的基于 Web 的用户界面。
用户界面在屏幕左侧（侧边栏）显示一个菜单栏，其中包含一组链接到 Web 界面不同功能的图标。

根据用户在登录用户界面时所拥有的角色，部分或全部菜单项将可用。
目标是只向用户展示与工作相关的条目，而不是将它们与其他不相关的条目混淆。

根据用户的角色，边栏中提供了以下条目。
